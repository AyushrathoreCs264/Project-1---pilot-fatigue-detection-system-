# ✈️ Pilot Fatigue Risk Detection System

## 📌 Overview

This project focuses on analyzing **pilot fatigue risk** using machine learning techniques.
It identifies potential fatigue conditions based on various **human factors** and predicts the risk level to improve aviation safety.

---

## 🎯 Objective

* To analyze human factors affecting pilot fatigue
* To apply machine learning models for prediction
* To improve aviation safety and decision-making

---

## 🧠 Human Factors Considered

* Sleep Duration
* Workload
* Stress Levels
* Heart Rate Variability
* Flight Hours
* Circadian Rhythm Disruption

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

## 🤖 Machine Learning Models Used

* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)

---

## 📊 Results

* Random Forest Accuracy: **80.59%**
* SVM Accuracy: **81.56%**
* KNN Accuracy: **78.92%**

👉 SVM performed the best.
